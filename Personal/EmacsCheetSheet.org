* Evil Mode (edición modal, estilo Vim)
** Modos
| *Modo*   | *Para qué*                 | *Entrar*                          |
|----------+----------------------------+-----------------------------------|
| Normal   | Moverte, borrar, comandos  | ~ESC~ desde cualquier otro modo   |
| Insert   | Escribir texto             | ~i~ ~a~ ~o~ ~O~                   |
| Visual   | Seleccionar texto          | ~v~ (carácter) ~V~ (línea)        |
| Replace  | Sobrescribir carácter      | ~R~                               |

** Movimiento (modo Normal)
| *Comando* | *Acción*                    |
|-----------+-----------------------------|
| ~h j k l~ | izq / abajo / arriba / der  |
| ~w~       | siguiente palabra           |
| ~b~       | palabra anterior            |
| ~e~       | fin de la palabra actual    |
| ~0~       | inicio de línea             |
| ~$~       | fin de línea                |
| ~gg~      | inicio del archivo          |
| ~G~       | fin del archivo             |

** Edición (modo Normal)
| *Comando* | *Acción*                          |
|-----------+------------------------------------|
| ~x~       | borra carácter bajo el cursor     |
| ~dd~      | borra la línea completa           |
| ~dw~      | borra hasta la siguiente palabra  |
| ~yy~      | copia (yank) la línea             |
| ~p~       | pega después del cursor           |
| ~u~       | deshacer                          |
| ~C-r~     | rehacer                           |
| ~3dd~     | borra 3 líneas (número + comando) |

* Leader Key (~SPC~) — Navegación general de Doom
| *Comando*   | *Acción*                          |
|-------------+------------------------------------|
| ~SPC SPC~   | ejecutar comando por nombre (M-x) |
| ~SPC .~     | buscar archivos (fuzzy finder)     |
| ~SPC f f~   | abrir archivo                      |
| ~SPC f s~   | guardar archivo                    |
| ~SPC b b~   | cambiar de buffer                  |
| ~SPC b d~   | cerrar buffer actual                |
| ~SPC p p~   | cambiar de proyecto                 |
| ~SPC w~     | comandos de ventanas (split/cerrar) |
| ~SPC o t~   | abrir terminal (vterm)              |
| ~SPC h d h~ | ayuda / documentación de Doom       |

* Localleader (~SPC m~) — Comandos según el modo actual
Cambia según si estás en org, LaTeX, elisp, etc. Explora con ~which-key~
(aparece solo, espera medio segundo tras presionar ~SPC m~).

* Git / Magit
| *Comando* | *Acción*             |
|-----------+-----------------------|
| ~SPC g g~ | abrir Magit (status)  |

Nota: ya NO usamos ~C-x g~ ni ~F10~ personalizados — nos quedamos con el
estándar de Doom (~SPC g g~) para no pelear contra el resto de sus atajos.

* Org-roam (cerebro digital, ~/Vault)
| *Comando*   | *Acción*                    |
|-------------+------------------------------|
| ~SPC n r f~ | buscar/crear nodo por título |
| ~SPC n r i~ | insertar link a otro nodo    |
| ~SPC n r c~ | capturar nueva nota rápida   |
| ~SPC n r t~ | buscar nodo por tag (custom) |
| ~C-c C-o~   | abrir enlace bajo el cursor  |

* LaTeX / Export a PDF
~F6~ guarda el org y lo exporta a PDF (ya NO abre Zathura automático)
~F7~ limpia archivos basura (aux, log, toc, etc)
~F8~ abre la carpeta del archivo actual
~C-c C-x C-l~ activa/desactiva la vista previa de una fórmula LaTeX puntual

| Comando anterior (vanilla) | Estado ahora           |
|-----------------------------+------------------------|
| ~C-c b~ (bloques de código) | igual, sigue activo    |
| ~C-c e l~ (bloque latex)    | igual, sigue activo    |
| ~C-c .~ (corrección palabra)| ahora vía flyspell/Doom|
| ~C-c i c~ (insertar cita)   | igual, sigue activo    |

* Reemplazo de palabras
~M-%~ "la palabra que queramos remplazar" ~RET~ "la nueva palabra" ~RET~

| Letra | Acción                          |
|-------+----------------------------------|
| ~y~   | remplazar ocurrencia             |
| ~n~   | saltar                           |
| ~!~   | remplazar todas sin preguntar    |
| ~q~   | salir                             |
| ~u~   | deshacer último remplazo         |
